# Web App Pages

This directory contains specifications for the web application pages.

## Pages

| # | Page | Description |
|---|------|-------------|
| 01 | Dashboard | Home page with stats, recent sessions, goals |
| 02 | Sessions | List of all workout sessions with filters |
| 03 | Movements | Library of movements with progressions |
| 04 | Calendar | Calendar/agenda view of workouts |
| 05 | Progressions | Unlock tracking and gamification |
| 06 | Stats | Detailed analytics and charts |
| 07 | Goals | Manage and track goals |
| 08 | Routines | Create and manage workout templates |

## Structure

Each page is documented in a numbered file:
- `01-dashboard.md`
- `02-sessions.md`
- `03-movements.md`
- `04-calendar.md`
- `05-progressions.md`

## Notes

- UI components shared across pages should be documented separately
- Design system (colors, typography) to be added later
- Responsive behavior to be documented per page
