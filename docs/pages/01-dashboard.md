# Dashboard

## Overview
The dashboard is the home page showing a summary of your training at a glance.

## Components

### Quick Stats Cards
- **This Week**: Number of sessions this week
- **Total Volume**: Volume trained this week
- **Current Streak**: Consecutive weeks with at least one session
- **Active Goals**: Number of active goals

### Recent Sessions
List of the 5 most recent workout sessions with:
- Date
- Duration (if tracked)
- Number of sets
- Tags

### Upcoming/Recent Goals
Shows progress on active goals with:
- Goal type and target
- Current progress
- Percentage complete

### Quick Actions
- **Start Workout**: Button to quickly start a new session
- **Log Weight**: Quick button to log bodyweight

### Bodyweight Chart
Small chart showing bodyweight trend for the last 30 days.
