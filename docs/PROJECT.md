# Workout Tracker Project

## Overview

This project is a workout tracking application consisting of:
- **Web App**: A web interface for users to track their workouts
- **CLI**: A command-line interface for agents to manipulate workouts

## Documentation

This directory contains **local project documentation**, not end-user documentation.

The documentation here is intended for:
- Developers working on the project
- Agents that need to understand the codebase
- Internal planning and design decisions

End-user documentation will be created separately when the project reaches a stable release.

## Structure

- `features/` - Feature documentation (domain concepts, CLI commands)
- `pages/` - Web app page specifications
